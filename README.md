# Home Page - Projeto Web

Este projeto foi desenvolvido como parte das aulas de Desenvolvimento Web.

A proposta foi criar uma home page com HTML e CSS que transmitisse confiança através do design e da organização das informações.

## Ideia

Escolhi o tema de viagens porque é algo que gosto e que transmite liberdade e experiências novas. A ideia foi criar um site simples, mas com uma mensagem clara.

## Decisões de design

- Cores em tons de azul foram usadas para transmitir calma e confiança
- Layout organizado em seções para facilitar a leitura
- Fonte simples para manter a clareza do conteúdo
- Imagem de fundo relacionada à praia para reforçar o tema

## Tecnologias utilizadas

- HTML
- CSS

## Observações

Este projeto foi feito com base no conteúdo aprendido nas primeiras semanas do curso, então a ideia foi manter um código simples, mas bem estruturado.
